Projects

For the moment, I don't have many projects, because I'm starting working as a web developer, but you can see my very first two projects. I hope to add more new ones in the future.

FIRESPOTTERS

This project consists of a predictor of wildfires' movement in real time. It uses 2 APIs to guess where a wildfire will develop and it shows it on the map with colorful cones. This was a project I made in 2023 NASA SpaceApps hackathon event and it won the data use prize and the best project.

PARCHIS

This was my last project in my first course of programming. It consists of a parchis game that can be played online by two players.
