FIRESPOTTERS

This project consists of a predictor of wildfires' movement in real time. It uses 2 APIs to guess where a wildfire will develop and it shows it on the map with colorful cones. This was a project I made in 2023 NASA SpaceApps Hackathon event and it won the data use prize and the best project.
